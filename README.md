# Rock-paper-and-scissors-game

🪨 Rock Paper Scissors Game (Python)
A simple command-line Rock Paper Scissors game built with Python.
The player plays against the computer which randomly selects its choice.
🎮 How to Play
Run the Python file
Type your choice:
rock
paper
scissors
The computer will randomly pick its move
The result will be displayed (Win / Lose / Draw)
Type quit to exit the game
🧠 Features
Random computer choices using Python random module
Continuous gameplay using a loop
Input-based user interaction
Win / Lose / Draw detection
▶️ Example

Choose rock, paper, or scissors (or type 'quit' to exit): rock
computer choose: scissors
you win
🛠️ Technologies Used
Python 3
random module
while loop
conditional statements (if/elif/else)
📌 Notes
This is a beginner Python project made for practice and learning basic programming concepts like loops, conditions, and randomness.
🚀 Author
Made by a beginner Python developer learning step by step.
